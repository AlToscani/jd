# jd: Java / RDBMS integration by JDBC

- Java SE 17
- JUnit 5 + AssertJ 3

## JDBC on
- MySQL 8
- Oracle DB 21c
- PostgreSQL 14
- SQLite 3

## Database setup
- SQLite
  - Run setup.sql to (re)set data

- MySQL, Oracle, PostgreSQL
  - Run once once.sql as dba to create user/schema "hron"
  - Run setup.sql as "hron" to (re)set data
